SLIDERS

1. Слайдер на Bootstrap:

  1. Требует подключения:
    • jQuery (jQuery 1.x / minified)
    • Bootstrap CSS
    • Bootstrap JS
  2. Плюсы:
    • высота картинок автоматически подстраивается;
    • можно добавлять любой контент вместо картинок;
    • простой в подключении, т.е просто подключи что надо и добавь картинок, подписей, индификаторов...
  3. Минусы:
    • есть скачки при разной высоте картинок, необходимо настраивать;
  4. Информация:
    • Glo Academy

2. Слайдер на Fotorama:

  1. Требует подключения:
    • jQuery
    • Fotorama CSS
    • Fotorama JS
  2. Плюсы:
    • Просто подключать;
    • Хорошо подходит для фотографий;
    • Можно подстроить под свои нужды (куча настроек, которые легко подключаются);
  3. Минусы:
    • Не обнаружены
  4. Информация:
    • Glo Academy

3. Слайдер на Slick:

  1. Требует подключения:
    • jQuery
    • Slick CSS
    • Slick JS
  2. Плюсы:
    • Просто подключать;
    • Под ваши нужды настраивается очень гибко;
  3. Минусы:
    • Что бы настроить слайдер необходимо использовать jQuery;
  4. Информация:
    • Glo Academy
